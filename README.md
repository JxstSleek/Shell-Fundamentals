# Bash To-Do List Manager

A simple interactive To-Do List Manager built with Bash scripting.

## Features

- View all tasks
- Add new tasks
- Delete tasks
- Stores tasks in `~/todo.txt`
- Interactive terminal menu
- Continuous loop until user exits

---

## How to Run

1. Clone the repository

```bash
git clone https://github.com/JxstSleek/shell-fundamentals.git
```

2. Navigate into the project directory

```bash
cd shell-fundamentals
```

3. Make the script executable

```bash
chmod +x todo.sh
```

4. Run the script

```bash
./todo.sh
```
